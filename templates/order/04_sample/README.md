# Sample

样稿产物放在这里。

建议文件：

```text
sample_brief.md
sample_slide_plan.md
sample_contract.json
sample_deck.pptx
sample_deck.pdf
sample_preview_images/
approved_sample_reference.json
style_master/
sample_qa.md
sample_delivery_message.md
```

`style_master/` 是正稿页面保持一致的真实视觉锚点。样稿通过后，正稿每页必须引用 approved sample 或 style master 里的图像参考，不能只靠文字描述风格。
