# Style Master

样稿通过后，把正稿必须复用的真实视觉锚点放在这里。

建议产物：

```text
style_anchor.png
template_master.png
navigation_bar_reference.png
cover_reference.png
section_reference.png
content_reference.png
data_reference.png
image_heavy_reference.png
style_spec.json
locked_elements.json
```

规则：

- `style_anchor.png` 是所有正稿页必须携带的主视觉参考。
- `template_master.png` 固定背景、边距、标题区、页脚、logo 区和导航条几何。
- `navigation_bar_reference.png` 必须单独给有导航条的页面。
- `style_spec.json` 记录颜色、字体层级、留白、圆角、阴影和图片处理方式。
- `locked_elements.json` 记录 logo、页码、导航条、页脚等固定元素坐标和高亮规则。
